🗂️ PROJETO 01 — CLI de Tarefas
📋 Regras do projeto

📝 Descrição do projeto
Crie um programa de linha de comando que permite ao usuário gerenciar uma lista de tarefas. Ao iniciar, o programa exibe um menu e espera o usuário escolher uma ação. O ciclo se repete até o usuário escolher sair.
O programa deve ter 5 funcionalidades:

Listar todas as tarefas (com id e status)
Adicionar uma nova tarefa (o usuário digita o título)
Concluir uma tarefa (o usuário informa o id)
Deletar uma tarefa (o usuário informa o id)
Sair do programa


💻 Exemplo de funcionamento
============================
     GERENCIADOR DE TAREFAS
============================
1. Listar tarefas
2. Adicionar tarefa
3. Concluir tarefa
4. Deletar tarefa
5. Sair
